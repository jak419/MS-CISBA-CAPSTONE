To Intitalize Git In the working folder
1. Initialize a Git Repository (if not already)

	cd ~/Capstone_CIDM-6395
	git init
	
2. Add a .gitignore File
	touch .gitignore
	
Add the following content:
	# Python cache
	__pycache__/
	*.pyc

	# Environment
	.env
	venv/
	myenv/
	.envrc

	# Jupyter Notebooks
	.ipynb_checkpoints/

	# Logs and outputs
	*.log
	*.csv
	*.png
	model_report.txt
	fraud_predictions*.csv
	roc_curve.png
	decision_tree_pipeline.joblib

	# IDEs
	.vscode/
	.idea/

3. Git is initiallized at /c/Users/johna/Capstone_CIDM-6395. Make sure you at this directory to make any git request.

4. Create a GitHub Repo (in browser)
Go to https://github.com/jak419

Click "New"

Name it something like fraud-detection-pipeline

Keep it public (or private), and don’t initialize with a README or .gitignore

Click "Create repository"

4. Add and Commit Your Files
	git add .
	git commit -m "Initial commit: Fraud detection pipeline with tests"
	git commit -m "Database integration commit: Fraud detection pipeline with tests"
	
5. Link Local Repo to GitHub Repo. Back in your terminal:
	git remote add origin https://github.com/jak419/MS-CISBA-CAPSTONE.git
	
6. Push to GitHub
git branch -M main  # rename master → main
git push -u origin main
